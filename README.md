# Solana Token Creator

A powerful Solana token generator and simulator built in Python. This tool allows users to easily create their own tokens on the Solana blockchain and provides simulation capabilities for testing.

## Features
- Complete Solana Token Generation logic
- Built-in simulation environment (`simulator.py`)
- Configurable environment options

## Setup & Execution
1. Install dependencies from `requirements.txt`:
   ```bash
   pip install -r requirements.txt
   ```
2. Set up your `.env` file with appropriate keys.
3. Run `main.py` or the `simulator.py` to get started!
